# ZZU_North_Card 校园卡灯板项目

ZZU_North_Card 是一个基于 ESP8266 芯片开发的智能校园卡灯板项目。它不仅是一款精美的装饰品，更集成了丰富的灯光效果、低功耗管理以及便捷的 Web 控制界面，旨在为你的校园卡增添科技感与个性化色彩。

## 项目特性/功能介绍

- **多种灯光模式**：支持常亮、脉冲、呼吸、彩虹渐变等多种预设灯光效果。
- **自定义色彩**：内置色轮模式，用户可以通过手机 Web 界面自由选取任意颜色。
- **智能电源管理**：
  - **自动休眠**：无操作 5 分钟后自动进入低功耗模式，关闭灯光后 30 秒自动休眠。
  - **按键唤醒**：通过侧边物理按键即可轻松唤醒系统。
- **便捷控制交互**：
  - **Web 控制台**：无需安装 App，连接热点即可通过手机浏览器调整颜色、模式和亮度。
  - **物理按键**：支持短按切换模式，长按无级调光。
- **OTA 在线升级**：支持通过 Web 界面无线上传固件（.bin 格式），方便功能迭代与更新。
- **线性亮度渐变**：开机时具备平滑的亮度渐变效果，提供更好的视觉体验。

## 硬件要求和连接说明

### 硬件清单
- **核心控制器**：ESP8266 (如 NodeMCU 或 ESP-12F 模块)
- **发光组件**：三色 RGB 灯（共阳极或共阴极，需根据代码调整）
- **按键**：一个物理轻触按键
- **电源**：3.7V 锂电池及充电电路（推荐）或 5V Micro-USB/Type-C 供电

### 引脚连接
| 组件 | ESP8266 引脚 (GPIO) | 说明 |
| :--- | :--- | :--- |
| **KEY_PIN** | GPIO 4 | 控制按键 |
| **LED_R** | GPIO 5 | RGB 灯红光控制 |
| **LED_G** | GPIO 12 | RGB 灯绿光控制 |
| **LED_B** | GPIO 13 | RGB 灯蓝光控制 |
| **LED_BUILTIN**| GPIO 2 | 板载提示灯 |

## 软件安装和配置指南

### 环境准备
1. 安装 [Arduino IDE](https://www.arduino.cc/en/software)。
2. 在 Arduino IDE 中添加 ESP8266 开发板支持：
   - 进入 `首选项` -> `附加开发板管理器网址`，添加 `http://arduino.esp8266.com/stable/package_esp8266com_index.json`。
   - 在 `开发板管理器` 中搜索并安装 `esp8266`。

### 固件烧录
1. 克隆本项目代码到本地。
2. 使用 Arduino IDE 打开 `main/main.ino`。
3. 在 `main/headers/config.h` 中可以根据需要修改默认的 WiFi 名称 (`ap_ssid`) 和密码 (`ap_password`)。
4. 将 ESP8266 连接至电脑，在开发板菜单中选择正确的型号。
5. 点击“上传”按钮进行烧录。

## 使用方法和操作说明

### 网络连接
1. 开启设备，设备将自动创建名为 `CampusLight` 的 WiFi 热点。
2. 使用手机连接该热点（默认密码：`12345678`）。
3. 大部分手机会自动弹出登录页面（强制门户），若未弹出，请在浏览器访问 `1.2.3.4`。

### 界面控制
- **模式切换**：点击页面上的“休眠”、“常亮”、“脉冲”等按钮进行切换。
- **亮度调节**：拖动滑块可实时改变灯板亮度。
- **颜色选取**：在色轮上点击或拖动以选择自定义颜色。
- **固件升级**：点击左上角的卫星图标进入 OTA 升级页面。

### 按键操作
- **短按**：在预设的几种模式间循环切换。
- **长按**：无级调节当前灯光的亮度。
- **休眠唤醒**：在设备休眠时，按下按键即可唤醒进入工作状态。

## 项目架构和技术细节

- **后端实现**：基于 `ESP8266WebServer` 实现轻量级 HTTP 服务器，处理来自前端的控制指令和状态请求。
- **前端技术**：采用原生的 HTML5、CSS3 和 JavaScript 开发。利用 `Canvas` 实现色轮取色，通过 `XMLHttpRequest` 与后端进行非阻塞通信。
- **核心逻辑**：
  - `core.cpp`: 负责系统状态管理、模式切换和非阻塞式动画效果更新。
  - `hardware.cpp`: 封装了底层的 PWM 调光控制和按键检测算法（包含软件消抖）。
  - `backend.cpp`: 处理 DNS 劫持实现强制门户功能，并响应 `/mode`, `/color`, `/bright` 等 API 请求。
- **能耗优化**：在空闲时段或进入休眠模式后，通过进入深度睡眠或关闭射频模块降低功耗，延长续航时间。

## 故障排除和常见问题

1. **连接 WiFi 后不弹出控制页面？**
   - 尝试在浏览器手动输入 `1.2.3.4` 或 `campus.light`。
2. **灯光颜色不准确？**
   - 请检查 `config.h` 中的引脚定义是否与实际硬件连接一致，以及 RGB 灯的极性是否正确。
3. **按键无法唤醒？**
   - 确保按键连接到了正确的 GPIO 引脚，并且连接了适当的上拉/下拉电阻（代码中默认开启了内部上拉）。
